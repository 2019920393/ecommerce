# 电商平台管理后台

基于 Vue3 + Element Plus + ECharts 的电商管理后台。

## 功能模块

| 模块 | 功能 |
|-----|------|
| 控制台 | 数据统计、订单趋势图、状态分布 |
| 商品管理 | 商品列表、新增/编辑、上下架 |
| 分类管理 | 分类树、新增/编辑子分类 |
| 订单管理 | 订单列表、详情、发货、取消 |
| 支付记录 | 支付流水查询 |
| 用户管理 | 用户列表、详情查看 |

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器 (端口5174)
npm run dev

# 构建生产版本
npm run build
```
